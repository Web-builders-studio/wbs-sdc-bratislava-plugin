Elementor-Widget fuer die SME Solution Day Connect 2026 Event-Landingpage (LinkedWorld AG, Bratislava, June 10-11).

== Description ==

Stellt ein Drag-und-Drop-Widget fuer Elementor bereit, das die komplette SDC-Bratislava-2026-Landingpage in eine bestehende WordPress-Seite einbettet. Self-contained React-Bundle, keine externen Dependencies. Auto-Updates via GitHub-Release.

Author: WBS Web Builders Studio (https://web-builders.studio)

== Installation ==

1. ZIP unter Plugins -> Hochladen installieren.
2. Aktivieren.
3. Beliebige Page mit Elementor oeffnen -> Widget "WBS — Solution Day Connect" platzieren.

== Frequently Asked Questions ==

= Welche Elementor-Version wird benoetigt? =

Mindestens Elementor 3.20.

= Funktioniert das Plugin ohne Elementor? =

Nein. Das Widget setzt Elementor voraus.
